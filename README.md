This branch contains a Python script to analyze character frequencies in a text file.

Features
Uses an abstract base class (BaseFrequencyAnalyzer) for frequency analysis.

Implements a concrete class (SimpleFrequencyAnalyzer) to:

Load a text file

Preprocess the text (remove special characters, normalize spaces)

Compute the frequency of alphanumeric characters

Return the six most frequent characters in the text

Usage
Update the file_path variable to point to your text file.

Run the script
